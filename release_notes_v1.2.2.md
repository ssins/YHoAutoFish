# YHo AutoFish v1.2.2 热修复版

v1.2.2 是基于 v1.2.1 的热修复版本，建议所有 v1.2 和 v1.2.1 用户升级。

## 本次修复

- 修复成功钓获时，画面偶发被误判为“鱼儿溜走了”，导致成功鱼被记录为空杆/失败的问题。
- 优化成功结算与失败提示的判定顺序，优先确认成功结算特征，再处理失败提示。
- 优化失败提示确认逻辑，低置信度提示不会立刻判定失败，会进行短暂连续确认，降低误判概率。
- 新增低分辨率启动提醒：游戏客户区低于 1600 × 900 时，会提示分辨率偏低并询问是否继续。

## 保留能力

- 保留 v1.2.1 对成功结算关闭、准备界面按钮误判、等待上钩流程的修复。
- 保留 v1.2 的高分辨率适配、溜鱼识别优化、失败恢复、捕获记录修复、用户接管保护和在线更新能力。
- 保留 GitHub 完整包更新与 Gitee 国内源分卷更新。
- 保留本地配置和捕获记录，升级不会清除 `config.json`、`records.json` 等用户数据。

## 升级建议

- v1.2 和 v1.2.1 用户建议升级到 v1.2.2。
- 如果正在钓鱼，请先停止自动钓鱼，再执行更新。
- 如果游戏分辨率低于 1600 × 900，建议先调高分辨率再开始自动钓鱼，以获得更稳定的识别效果。

## 下载文件说明

- GitHub 用户下载 `YHoAutoFish-v1.2.2-windows.zip`。
- Gitee 国内源会提供 `YHoAutoFish-v1.2.2-windows.zip.001`、`.002` 等连续分卷。
- 程序内一键更新会自动处理完整包或分卷包，并在安装前校验文件完整性。
